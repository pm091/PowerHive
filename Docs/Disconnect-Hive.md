# Disconnect-Hive

## SYNOPSIS
Disconnects Hive Session

## SYNTAX

```
Disconnect-Hive [<CommonParameters>]
```

## DESCRIPTION
Deletes Hive Session and clears the $Global:HiveHead variable

## EXAMPLES

### EXAMPLE 1
```
Disconnect-Hive
```
