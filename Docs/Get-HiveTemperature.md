# Get-HiveTemperature

## SYNOPSIS
Gets current hive temperature

## SYNTAX

```
Get-HiveTemperature [<CommonParameters>]
```

## DESCRIPTION
Gets the current hive temperature and reported time values

## EXAMPLES

### EXAMPLE 1
```
Get-HiveTemperature
```
