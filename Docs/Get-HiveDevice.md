# Get-HiveDevice

## SYNOPSIS
Retrieves Hive Devices

## SYNTAX

```
Get-HiveDevice [-detailed] [<CommonParameters>]
```

## DESCRIPTION
Retrieves Hive Devices

## EXAMPLES

### EXAMPLE 1
```
Get-HiveDevice
```

### EXAMPLE 2
```
Get-HiveDevice -Detailed
```
