# Get-HiveAsscociatedDevice

## SYNOPSIS
Gets Linked Management Devices

## SYNTAX

```
Get-HiveAsscociatedDevice [<CommonParameters>]
```

## DESCRIPTION
Retrieves list of devices where the hive management app is installed
